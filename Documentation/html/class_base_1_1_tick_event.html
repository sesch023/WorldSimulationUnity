Detailed Description

TickEvent which is triggered at a certain tick number once. Needs to be removed after it has been triggered, by giving a proper OnRemoval delegate. UpdateManager deals with triggering and will automatically call managed removal of the TickEvent. Automatically registers itself with the UpdateManager.
