Detailed Description

Controller MonoBehaviour for a Camera. It requires a Player Input which messages the methods OnMoveHorizontal, OnMoveVertical and OnZoom. It then controls the required Camera Component, which is clamped to the TileMap given in the MapController of the MapManager.
